Valky Stream Service provides more than 1000 Movies and 250 Series, all that in HD quality!

Valky Cloud Service
The VCS is a custom made cloud platform, created and hosted with Windows Server Standard.
It uses some of the built in roles and features, but got modfied. You can think about it like GoogleDrive or DropBox.
You can choose a folder on your computer to sync it with VCS, use the web client for quick access or use an app on your smartphone.
